# Riemann Sums

Calculate and visualize Riemann sums in the web browser at http://tntran09.github.io/riemann

##### Technologies
React, Bootstrap, D3, Flux, Browserify

## Contributing
### Install

```
npm install -g electron-prebuilt watchify browserify
npm install --production
```

### Build
1) Bundle scripts
```
npm run watchify
```
2) Run electron
```
npm start
```
